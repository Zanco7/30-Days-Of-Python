This project is a command-line Flash Card Learning App built with Python as part of my **30-Day Python Challenge**.

## Features
- Add custom flashcards (question & answer)
- Study mode to review all flashcards
- Quiz mode with score tracking
- View all saved flashcard questions
- Simple menu-driven interface

## How It Works
1. Users can add flashcards by entering a question and its answer.
2. Study mode displays all questions with their answers.
3. Quiz mode randomly asks questions and checks user answers.
4. The score is tracked during quiz sessions.
5. The app continues running until the user exits.

- User input handling
