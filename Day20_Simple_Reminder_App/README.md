This project is part of my **30-Day Python Challenge**.

The Simple Reminder App allows users to set a reminder message that will be displayed after a specified number of seconds. The program validates user input and allows setting multiple reminders in one session.

## Features
- Accepts a custom reminder message
- Accepts reminder time in seconds
- Input validation for empty messages and invalid time
- Option to set multiple reminders
- Uses system delay to trigger reminders
